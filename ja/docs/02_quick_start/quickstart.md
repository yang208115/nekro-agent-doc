# クイックスタート

Nekro Agent `(以下、NA)` へようこそ！このドキュメントは、この強力なAIエージェントシステムを素早く使い始めるのに役立ちます。

::: warning ⚠ セキュリティ警告 ⚠

このプロジェクトは、AIが実行時に隔離されたDockerコンテナ環境内で任意のコードを実行することを許可します。アプリケーションはコンテナ隔離や権限制限など複数のセキュリティ対策を採用していますが、以下のようなセキュリティリスクが依然として存在する可能性があります：

1. IPアドレスの漏洩
2. コンテナからのエスケープ
3. その他未知のリスク

! リスクを認識し、自己責任で負担することを承諾した上でのみ続行してください。作者は、このプロジェクトの使用によって生じたいかなる損失についても責任を負いません！
:::

## 公式デプロイオプション

オペレーティングシステムに基づいて適切なデプロイ方法を選択してください：

- [Linux デプロイチュートリアル](/ja/docs/02_quick_start/deploy/linux) - apt をパッケージマネージャーとして使用している Linux ディストリビューション向け
- [Windows デプロイチュートリアル](/ja/docs/02_quick_start/deploy/windows) - Windows 10/11 システムに適しています
- [MacOS デプロイチュートリアル](/ja/docs/02_quick_start/deploy/macos) - MacOS システムに適しています

## コミュニティデプロイソリューション

- [Nekro-Agent-Toolkit](/ja/docs/community/nekro-agent-toolkit) クロスプラットフォームデプロイCLIツール
  - **クロスプラットフォーム** デプロイソリューションであり、NekroAgentのための便利な **インストール、バックアップ、リカバリー** ソリューションです。

- [1Panel アプリケーションデプロイ](https://github.com/lgc2333/nekro-agent-deploy-1panel)

  - **1Panel パネル** に基づくすぐに使えるワンクリックデプロイソリューション。純粋なインターフェースデプロイ操作とOpenRestyリバースプロキシ構成をサポートし、NapCatプロトコルエンドポイントを自動的に構成します。

- [iStoreOS システムデプロイ](/ja/docs/community/iStoreOS)

  - **OpenWRT** に基づくソフトルーターシステム上でNekro Agentを実行し、ワンクリックインストールスクリプトを提供します。

## 基本設定

デプロイが成功した後、以下の基本設定を完了する必要があります：

1. **プロトコル設定**: 選択したチャットプラットフォームに接続します
2. **モデル設定**: AIモデルとAPIキーを設定します
3. **システム設定**: 基本的なシステムパラメータとユーザー権限を設定します

::: warning ⚠ 重要なセキュリティ情報 ⚠
セキュリティ上の理由から、以下の情報を必ずお読みください：

1. デフォルトのプロトコルパスワードを必ず変更してください：パブリックWebUIは強力なパスワード（数字、文字、記号を含む12文字以上推奨）を使用する必要があります
2. セキュリティ上の理由から、デフォルトのポートの使用は避けてください
   :::

詳細な設定手順については、以下のドキュメントを参照してください：

- [プロトコル設定](/ja/docs/02_quick_start/config/protocol)
- [システム設定](/ja/docs/02_quick_start/config/system)
- [アプリケーションの更新](/ja/docs/02_quick_start/config/update)

## 初めての使用

### 管理インターフェースへのアクセス

デプロイ完了後、以下のアドレスから管理インターフェースにアクセスできます：

- `http://<サーバーIP>:8021` (リモートアクセス)
- `http://localhost:8021` (ローカルアクセス)

### 管理インターフェースへのログイン

初回ログイン時は、デプロイプロセス中に生成された管理者アカウントとパスワードを使用してください

### 基本的な使用フロー

1. **プロトコル接続**: プロトコル設定を完了し、接続状態を確認します
2. **ペルソナ設定**: AIの基本的なペルソナと行動パターンを設定します
3. **会話テスト**: 選択したプラットフォームを通じてAIとの会話をテストします
4. **機能拡張**: 必要に応じて希望する拡張機能を有効にします

## 次のステップ

基本設定完了後、より高度な機能を探索できます：

- [モデルグループ設定](/ja/docs/03_advanced/model_config) - 複数のモデルの組み合わせを設定します

##クイックリファレンス

### 一般的な管理コマンド

```bash
# NAコンテナログを表示
sudo docker logs -f nekro_agent

# NapCatコンテナログを表示
sudo docker logs -f nekro_napcat

# サービスを再起動
sudo docker restart nekro_agent
```

### 一般的なチャットコマンド

```
/na-info - 現在のアプリケーション情報を表示
/na-help - ヘルプ情報を表示
/reset - 現在のセッションをリセット
```
